This is my personal blog built with [Hugo](https://gohugo.io/), a fast and flexible static site generator.

## Overview

- **Framework**: Hugo
- **Hosting**: Nginx in Docker on ThinkCentre
- **Domain**: diogoqueiroz.page/blog via Cloudflare tunnels

## Local Development

To run the blog locally:

1. Install Hugo (see [Hugo installation guide](https://gohugo.io/getting-started/installing/))
2. Clone this repository
3. Run `hugo server` in the project directory
4. Open `http://localhost:1313` in your browser

## Deployment

The site is hosted on a ThinkCentre using Nginx in Docker and accessed via Cloudflare tunnels at [diogoqueiroz.page/blog](https://diogoqueiroz.page/blog).

For deployment details, refer to the Docker and Cloudflare configurations in the repository.
